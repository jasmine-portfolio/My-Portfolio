Alertly: Widget Display

Aug 2024 - Present

#diy-project #esp32 #productivity

I love using the widgets on my phone but they get buried under a sea of apps and pages. To help manage that, I created Alertly, a productivity-focused device designed to provide quick, accessible reminders conveniently on the workspace. With a dedicated display, it features essential widgets like a clock, calendar, to-do list, and countdown timer. The goal is to keep organized and mindful of time, decreasing distractions often caused by apps and multiple open pages on your phone. Alertly is powered by 7.5’’ e-paper display and ESP32 microcontroller with build in Wifi, all programmed on Arduino IDE.

Line Following Robot with Remote Control

Nov - Dec 2024

#bcit-group-project #ti #robotics

Line Following Robot with Remote Control project integrates the TI TMS320F28379D microcontroller and an ESP32 module for wireless control. The robot utilizes a PID control algorithm to calculate error to adjust its speed in real-time, enabling it to follow a line autonomously. The ESP32 module allows remote adjustments of settings, including motor speed, via Wi-Fi and SCI protocol. Built on TI BIOS, the system efficiently manages real-time tasks using threads and interrupts for optimized performance.

Color Sorter

Mar 2023

#bcit-project #raspberry-pi #automation

This is an automatic 4-color ball sorter that efficiently sorts up to 16 balls in just 15 seconds. Powered by a Raspberry Pi, it utilizes computer vision, servo motors, and LED feedback for real-time color detection and sorting. Featuring GPIO control, the system uses a simple GUI for manual operation in C++, allowing users to monitor and control the sorting process.
