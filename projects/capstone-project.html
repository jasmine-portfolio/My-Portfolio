Overview

Hospital environments can be intimidating and even isolating, making it difficult for children to cope with the healthcare setting. To address this, we will customize the MiRo-E robot, an interactive companion, to engage young patients in therapeutic activities like guided breathing exercises, interactive games, and relaxation techniques, helping to ease anxiety and improve their hospital experience. The robot is also equipped with the ability to recognize QR/ARUCO codes, which trigger various activities, making it a versatile tool for both patients and healthcare professionals.

Concept

MiRo-E is an interactive social robot is going to include the following key features:

  • Guided Breathing Exercises: Calming techniques to manage anxiety.
  • Interactive Play: Engaging games to distract and entertain.
  • QR/ARUCO Code Recognition: Easy activity initiation via scanning codes.
  • Sensor Integration: Tailored interactions based on touch and motion.

Developed with ROS, Ubuntu, and Python, MiRo-E uses real-time sensors and custom programming to enhance children's hospital experiences.

Achievements

  • Improved recognition speed and accuracy using OpenCV and ROS, reducing errors and enhancing system reliability for activity triggers.
  • Designed child-friendly interfaces with visual cues and audio feedback to improve accessibility and engagement.
  • Optimized system coordination by dividing it into two distinct processes for smooth communication and transitions.
  • Combined multiple sensor types and conducting extensive real-time testing, refining user interactions for consistent performance.

Progress

We have developed two activities that still require some refinement: a guided breathing exercise and an interactive audiobook. Eventually, we aim to incorporate a game and enhance the movement sequences to make them feel more natural.
