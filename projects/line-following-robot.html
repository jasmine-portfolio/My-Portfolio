Overview

Line Following Robot with Remote Control project integrates the TI TMS320F28379D microcontroller and an ESP32 module for wireless control. The robot utilizes a PID control algorithm to calculate error to adjust its speed in real-time, enabling it to follow a line autonomously. The ESP32 module allows remote adjustments of settings, including motor speed, via Wi-Fi and SCI protocol. Built on TI BIOS, the system efficiently manages real-time tasks using threads and interrupts for optimized performance.

Concept

The robot autonomously follows a line using PID control based on real-time sensor feedback, while transmitting RPM and power consumption data to a remote control for monitoring. The system is designed for:

  • SYS/BIOS RTOS which manages multiple thread types:
    • HWI handles timers for periodic tasks and XINT interrupts for encoder speed and serial data.
    • SWI performs data calculations, including PID control and data transmission, based on HWI data.
    • Idle State controls a heartbeat LED to indicate system activity.
  • CPU efficiency and low power by leveraging threaded interrupts and periodic functions.
  • Standalone operation with ESP32 for wireless communication via Wi-Fi UDP and SCI (UART) protocol.

Achievements

  • Refined line-following precision through iterative testing of PID constant adjustments for smooth responsive navigation.
  • Developed a protocol and optimizing system performance to minimize transmission delays.
  • Balanced accurate power consumption measurements without affecting performance for monitoring system.
  • Tested motors, sensors, and microcontrollers for smooth operation, ensuring reliable system performance.
  • Implemented error-checking to maintain stability and prevent critical failures (e.g., divide-by-zero errors).

Outcome

The project successfully created a line-following robot with remote control, real-time performance monitoring, and adjustable settings. The robot can follow a line autonomously, with feedback on speed and power consumption, offering greater flexibility and control.
